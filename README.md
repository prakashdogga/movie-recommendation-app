📌 Project Overview

This project implements a content-based movie recommendation system.
It suggests movies similar to the one selected by the user.

The application includes:

A Machine Learning model for movie similarity

A Flask web application

A MySQL database for storing user search history

A simple web interface

🚀 Features

🎥 Recommend similar movies

🔍 Search for any movie in the dataset

💾 Store user searches in MySQL database

🌐 Web interface built with Flask

⚡ Fast recommendations using precomputed similarity matrix

🛠️ Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* NumPy
* MySQL

HTML / CSS

Pickle (for saving ML model)
